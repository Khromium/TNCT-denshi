1.課題
8bitSWの1がOFFの時，0.5秒周期で２つのLEDを同時点滅，
8bitSWの1がONの時，1秒周期で２つのLEDを同時点滅


2.使用したプログラム
 /**********************************************************
8ビットスイッチによってLEDのON-OFFを行う
**********************************************************/
#include <3048fone.h>
#include "h8_3048fone.h"

void waitmsec(int msec)
/*msec間なにもしない時間稼ぎ関数*/
{
    int i,j;
    for (i=0;i<msec;i++) {
        for (j=0;j<4190;j++);    /*4190は実測によって求めた値 25MHz駆動*/
    }
}

main()
{
    initLed();
    init8BitSW();/*8bitSWの初期化*/
    while(1) {
        if (!check8BitSW(1)) { /*8bitSWの1がOffの時*/
            turnOnLed(0);
            turnOnLed(1);
            waitmsec(250);
            turnOffLed(0);
            turnOffLed(1);
            waitmsec(250);
        }else if (check8BitSW(1)) { /*8bitSWの1がOnの時*/
            turnOnLed(0);
            turnOnLed(1);
            waitmsec(500);
            turnOffLed(0);
            turnOffLed(1);
            waitmsec(500);
        }
    }
}
3.考察
場合分けし，点灯時間を調整することで課題を実現させることができた．
